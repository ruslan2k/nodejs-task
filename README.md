# URL Shortener

## Run

```
# In docker
$ make docker

# With node.js
$ make
```

## Access

[http://localhost:3000](http://localhost:3000)

Statistics: [http://localhost:3000/shorturls](http://localhost:3000/shorturls) admin / nimda
